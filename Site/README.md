# AudieVit — Sito di documentazione

Sito web di documentazione per AudieVit, realizzato con Angular 21.

## Sviluppo

```bash
ng serve
```

Naviga su `http://localhost:4200/`. L'applicazione si ricarica automaticamente.

## Build

```bash
ng build
```

I file generati sono in `dist/Site/`.

## Test

```bash
ng test
```

## Struttura

```
src/app/
├── header/           # Barra di navigazione
├── sidebar/          # Menu laterale con sezioni
├── doc-content/      # Contenuto della documentazione
├── toc/              # "On This Page"
└── footer/           # Piè di pagina
```
