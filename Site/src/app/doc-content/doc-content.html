Getting Started

Welcome to AudieVit! This guide will help you set up and run the voice assistant.

Installation

Clone the repository and install dependencies:

git clone https://github.com/FrankStrix/AudieVit.git
cd AudieVit/App
npm install
npm run dev

Open http://localhost:1420 in your browser.

Quick Start

Once the app is running, click "Start Listening" and grant microphone access. The TensorFlow.js model will load, and you can start speaking commands.

Voice Recognition

AudieVit uses the Speech Commands dataset by Google, running entirely in-browser via TensorFlow.js.

How It Works

The app captures audio from your microphone, runs it through a pre-trained neural network, and classifies the spoken word. No data leaves your machine.

TensorFlow Setup

TensorFlow.js is loaded via CDN in index.html. The speech-commands model runs with the BROWSER_FFT backend.

Supported Commands

  • yes / no — confirmation
  • up / down / left / right — directions
  • stop / go — actions
  • on / off — toggles
  • zero — nine — numbers

API Reference

The code is split into simple service files under src/services/.

Audio Service

Handles microphone access via getUserMedia. Exposes getMicrophone() and stopMicrophone().

Classifier

Wraps the speechCommands.create('BROWSER_FFT') model. Provides loadModel(), startListening(), and stopListening().

Text To Speech

Uses the Web Speech API (speechSynthesis) to speak responses. Simple speak(text) and stopSpeaking() functions.
