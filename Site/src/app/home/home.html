AudieVit

Il tuo assistente vocale AI, locale e privato.

AudieVit è un assistente vocale AI che gira interamente nel tuo browser, senza inviare dati a server esterni. Sfrutta le API Web Speech per il riconoscimento e la sintesi vocale, e si connette a un'istanza locale di Ollama per generare risposte intelligenti.

Riconoscimento vocale

Trascrizione in tempo reale tramite Web Speech API.

🤖

Risposte AI

Generazione tramite Ollama con modello llama2 in locale.

🔊

Sintesi vocale

Lettura ad alta voce delle risposte con voci naturali.

🌍

Multi-lingua

Rilevamento automatico e risposta in IT, EN, FR, ES, DE.

⚡

Streaming

Risposte in tempo reale carattere per carattere.

🔒

Privacy prima di tutto

Tutto il processing avviene in locale. Nessun dato lascia il tuo computer.
